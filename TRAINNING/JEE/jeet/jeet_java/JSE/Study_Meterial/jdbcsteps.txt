Class.forName("com.mysql.jdbc.Driver");

ii. After Getting Driver class,jvm will load Byte code into memory,at the loading time a static block will execute that will call DriverManager.RegisterDriver() method.
( FROM JAVA 7 IT IS NOT REQUIRED )
ii. Appropite Driver is setup for our JDBC application.

2. Establish Connection between Java & DB
Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/studentdb","root","abc123");

jdbc - main protocol
mysql://localhost:3306 - subprotocol
studentdb - database

getconnection internally call Driver.connect() method
establish socket connection
